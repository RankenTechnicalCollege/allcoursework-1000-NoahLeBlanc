Network Protocols

HTTP
HTTP stands for "HyperText Transfer Protocol" which is an underlying Network Protocol allowing the transfer of Hypermedia docs on the web!
   More info here!
FTP
FTP stands for File Transfer Protocol it is an Insecure Protocol for transferring files from one host to another over the web. but you do require an FTP client in order to do this.
One good example of a FTP client is FileZilla. However there are many more to choose from some IDEs even have built in FTP clients
More Info here!
HTTPS
HTTPS stands for HyperText Transfer Protocol Secure which is simply an encrypted version of the HTTP Protocol it uses TLS Transport Layer Security to encrypt info between clients and servers
